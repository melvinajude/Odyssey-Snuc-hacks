# Odyssey FinTech

Welcome to Odyssey - The next-generation banking platform for the modern economy. Built during the Odyssey Snuc hacks hackathon!

## Features
- **Smart Dashboard**: View your balances and transactions with real-time updates.
- **Modern Interface**: A beautiful glassmorphism dark-mode UI.
- **Lightning Fast**: Built on lightning-fast technologies.

## How to run
Simply open `index.html` in your web browser. Nothing to compile or install!

### Hackathon Submission
This project is built using HTML, CSS, and Vanilla JavaScript with a custom CSS styling engine taking inspiration from modern FinTech applications.
